# LendingClub-DefaultPrediction
Predicting loan outcomes using logistic regression, random forest, and XGBoost models 
This project builds an ensemble model (logistic regression, random forest, and XGBoost) to predict loan outcomes from LendingClub data.  
It demonstrates:
- Data cleaning and leakage detection  
- Automated hyperparameter tuning  
- Model evaluation and visualization  

**Tools:** Python, pandas, scikit-learn, XGBoost, matplotlib
